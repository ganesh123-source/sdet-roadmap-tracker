\# SDET OS — Roadmap Tracker



A dynamic, interactive roadmap tracker designed for aspiring Software Development Engineers in Test (SDETs). This dashboard helps track progress from basic Python fundamentals all the way to CI/CD and DevOps awareness.



\## 🚀 Features

\* \*\*Interactive Progress Tracking:\*\* Mark topics as complete and watch your overall progress bar update in real-time.

\* \*\*Phase-by-Phase Breakdown:\*\* Covers Python, Core Programming, Testing Fundamentals, Manual Testing, Automation (Selenium/PyTest), API Testing, Git, and CI/CD.

\* \*\*Full Timeline View:\*\* Switch to the timeline tab to see the estimated timeframes to become job-ready (\~5 months).

\* \*\*Cloud Sync (Optional):\*\* Built-in support to integrate with Supabase so your progress saves across all devices.



\## 🛠️ Tech Stack

\* HTML5 / CSS3 (Grid \& Flexbox)

\* Vanilla JavaScript

\* \[Supabase](https://supabase.com/) (Backend-as-a-Service for user auth and database sync)



\## 💻 How to Run Locally

1\. Clone the repository to your local machine.

2\. Open the `index.html` file directly in any modern web browser.

3\. \*Optional:\* To enable cloud saving, replace the Supabase URL and Anon Key at the top of the HTML file with your own credentials.
